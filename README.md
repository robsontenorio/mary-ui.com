## Contributing

This website is made with Laravel, Livewire Volt and maryUI.

Clone the repository.

```bash
git clone git@github.com:robsontenorio/mary-ui.com.git
```

Create the `.env` from `.env.example`.

Install, migrate and start.

```bash
composer start
```
